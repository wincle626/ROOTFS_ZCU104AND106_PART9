TeX Live zaprasza

TeX Live oferuje pełny system TeX-a dla wielu różnych systemów operacyjnych, Unix, Macintosh, Windows i wielu innych. Znajdziemy tu programy do składu tekstów, programy do przeglądania i drukowania dokumentów TeX-owych w wielu językach oraz bogate kolekcje makr i fontów.

Dystrybucja zawiera obszerną dokumentację TeX-a, a także dokumentacje towarzyszące poszczególnym pakietom oprogramowania.

Infomacje licencyjne (w języku angielskim) można znaleźć w plikach tekstowych LICENSE.TL oraz LICENSE.CTAN.

Od czego zacząć

Przed rozpoczęciem instalacji zalecamy wydrukować i przeczytać Przewodnik po TeX Live. Prowadzi on krok po kroku przez instalację, znajdziemy tam opis głównego drzewa katalogów i dużo więcej.

Gdy instalujemy z płytki DVD, wersje Przewodnika w kilku językach dostępne są w podkatalogu texlive-doc.

Po zainstalowaniu systemu, Przewodnik w języku angielskim w formacie html i pdf znajdziemy w texmf-dist/doc/texlive/. Tam też są dostępne inne wersje językowe Przewodnika, w tym polska.

Dołączyliśmy również listę odnośników do dokumentacji HTML and PDF programów i pakietów zawartych w instalacji.

Jeśli instalujemy TeX Live z internetu, powyższe odnośniki nie działają. W tym przypadku należy sięgnąć do dokumentacji tutaj: TeX Live documentation on the Internet.

Lista najważniejszych zmian w bieżącej edycji znajduje się na końcu dokumentacji w rozdziale Historia.

Instalacja

Skrypt instalacyjny nazywa się install-tl (install-tl-windows.bat dla Windows) i ma dwa tryby pracy:

-gui
(domyślny dla Windows i Macs), który używając interfejsu graficznego, bez zbędnych zapytań instaluje cały system, ale pozwala również na wybranie instalacji zaawansowanej.

-no-gui
(domyślny dla wszystkich innych systemów), który w trakcie instalacji umożliwia wybór wszystkich dostępnych opcji za pomocą prostego interfejsu tekstowego.

Do dodawania/usuwania pakietów, aktualizacji z internetu oraz innych czynności konfiguracjnych można użyć programu tlmgr (TeX Live Package Manager), który tak jak program instalacyjny, ma dwa tryby pracy (graficzny oraz tekstowy).

Szczegółowe informacje o install-tl i tlmgr uzyskamy uruchamiając je z linii poleceń z parametrem --help. Więcej informacji można znaleźć w dokumentacji, o której była mowa powyżej.

Przydatne linki

Na koniec kilka adresów stron internetowych, na których można znaleźć więcej informacji o TeX-u. Nie są one dołączone do TeX Live:

  • tug.org/begin.html - dla początkujących użytkowników TeX-a i/lub potrzebujących podstawowych o nim informacji.
  • tug.org/interest.html - główna lista internetowych zasobów TeX-owych, systemów, pakietów, fontów i wielu innych, także dokumentacji.
  • tug.org/ctan.html - jeden z adresów Comprehensive TeX Archive Network (CTAN), który zawiera światowe archiwa TeX-owe.
  • tug.org/usergroups.html - lista grup użytkowników systemu TeX na całym świecie, wraz z danymi kontaktowymi. Gorąco zachęcamy do przyłączenia się do wybranej grupy użytkowników, a także wspomagania prac przy TeX Live oraz innych aktywności.
